# Dev Notes

Written in markdown and built using mkdocs.

## Project layout

    mkdocs.yml    # The configuration file.
    docs/
        index.md  # The documentation homepage.
        ...       # Other markdown pages, images and other files.

To run:
```
mkdocs serve
```
Access your site:
http://127.0.0.1:8001/
