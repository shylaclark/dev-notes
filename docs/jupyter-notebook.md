# Jupyter Notebook

## Shortcuts
- Command mode:  `Ctrl` + `Enter`
- Edit mode:  `Enter`
- Back to command mode: `Esc`
- Insert cell above: `A`
- Insert cell below: `B`
- Autocomplete: `Tab`
