# npm and Yarn

## npm

### Getting Started

package.json
package-lock.json

#### Install dependencies
```
$ npm install
```

#### Update only what has changed?
```
$ npm update
```

#### Start the server (e.g. localhost:8080)
```
$ npm start
```

#### Remove and reinstall all the dependencies
```
$ rm -rf node_modules/
$ npm install
```
